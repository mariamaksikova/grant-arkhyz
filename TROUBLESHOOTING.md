## Проверка 1: Запущен ли сервер?

Откройте терминал и выполните:
```bash
cd /Users/mariamaksikova/Documents/VSCODE/архыз/grant-arkhyz

```

Вы должны увидеть:
```

Таблица reviews готова
Таблица investments готова
```

## Проверка 2: Тест базы данных

Выполните:
```bash
node test-server.js
```

Это покажет:
- Какие таблицы существуют
- Сколько записей в каждой таблице
- Последние 5 бронирований

## Проверка 3: Проверка в браузере

1. Откройте сайт: `http://localhost:3000`
2. Откройте консоль разработчика (F12 или Cmd+Option+I)
3. Заполните и отправьте форму
4. В консоли должны появиться сообщения:
   - "Отправка данных формы: ..."
   - "Статус ответа: 200"
   - "✓ Данные успешно отправлены"

## Проверка 4: Проверка в терминале сервера

При отправке формы в терминале сервера должны появиться:
```
Получен запрос на /api/submit-form
Тип формы: booking (или review, или investment)
Данные: {...}
✓ Бронирование сохранено с ID: 1
```

## Возможные проблемы:

### Проблема: "Сервер недоступен"
**Решение:** Убедитесь, что сервер запущен на порту 3000

### Проблема: CORS ошибка
**Решение:** Убедитесь, что в server.js есть `app.use(cors());`

### Проблема: Данные не сохраняются, но ошибок нет
**Решение:** 
1. Проверьте права на запись в папку проекта
2. Проверьте, что база данных database.db существует
3. Запустите `node test-server.js` для проверки

### Проблема: Ошибка "Cannot find module"
**Решение:** Выполните `npm install` для установки зависимостей

## Проверка данных в базе

Откройте админ-панель: `http://localhost:3000/admin.html`

Или используйте SQLite команды:
```bash
sqlite3 database.db
.tables
SELECT * FROM bookings;
SELECT * FROM reviews;
SELECT * FROM investments;
.quit
```
