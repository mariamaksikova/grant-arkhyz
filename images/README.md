# Папка для изображений

## Как добавить изображения:

1. **Основные изображения для сайта:**
   - `hero-bg.jpg` - фоновое изображение для главной секции (рекомендуемый размер: 1920x1080px)
   - `about-image.jpg` - изображение для секции "О турбазе"
   - `nature-1.jpg`, `nature-2.jpg`, `nature-3.jpg` - изображения для секции "Природа и развлечения"
   - `block-13x10.jpg` - изображение блока 13,5x10 (3 этажа, 15 номеров)
   - `block-10x9-2fl.jpg` - изображение блока 10x9 (2 этажа, 6 номеров)
   - `block-10x9-1fl.jpg` - изображение блока 10x9 (1 этаж, 3 номера)
   - `room-interior.jpg` - интерьер номера
   - `bathhouse.jpg` - баня с бассейном
   - `fire-zone-cinema.jpg` - зона костра с кинотеатром
   - `fire-zone-swings.jpg` - зона костра с качелями
   - `gazebo.jpg` - беседка
   - `location-map.jpg` - карта расположения

2. **Формат изображений:**
   - Используйте формат JPG или PNG
   - Оптимизируйте изображения для веб (можно использовать онлайн-инструменты для сжатия)
   - Рекомендуемый размер для фоновых изображений: 1920x1080px
   - Рекомендуемый размер для карточек: 800x600px

3. **После добавления изображений:**
   - Убедитесь, что имена файлов совпадают с теми, что указаны в HTML
   - Если вы используете другие имена, обновите пути в файлах index.html и start-sales.html
