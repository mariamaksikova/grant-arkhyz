# Быстрый старт

## 1. Запуск сервера

Откройте терминал в папке проекта и выполните:

```bash
npm start
```

Вы должны увидеть:
```
==================================================
✓ Сервер запущен на http://localhost:3000
✓ База данных: /path/to/database.db
==================================================

Ожидание запросов...
```

## 2. Откройте сайт

**ВАЖНО:** Откройте сайт через браузер по адресу:
```
http://localhost:3000
```

**НЕ открывайте файлы напрямую через file://** - формы не будут работать!

## 3. Проверка работы

1. Откройте консоль браузера (F12 или Cmd+Option+I)
2. Заполните любую форму
3. В консоли должны появиться логи:
   - "Отправка данных формы: ..."
   - "Статус ответа: 200"
   - "✓ Данные успешно отправлены"

4. В терминале сервера должны появиться:
   - "Получен запрос на /api/submit-form"
   - "✓ Бронирование сохранено с ID: 1"

## 4. Проверка сохраненных данных

Откройте админ-панель:
```
http://localhost:3000/admin.html
```

Или проверьте через терминал:
```bash
node test-server.js
```

## Валидация форм

Все формы теперь имеют валидацию:
- **Имя**: минимум 2 символа, только буквы
- **Телефон**: формат +7 999 123-45-67 или 8 999 123-45-67
- **Email**: корректный формат email
- **Даты**: дата выезда должна быть позже даты заезда

## Тестирование форм

Откройте тестовую страницу для проверки работы всех форм:
```
http://localhost:3000/test-forms.html
```

Эта страница позволит проверить:
- Доступность сервера
- Отправку отзывов
- Отправку бронирований

## Если не работает

1. Убедитесь, что сервер запущен (`npm start`)
2. Убедитесь, что открываете сайт через `http://localhost:3000`
3. Откройте консоль браузера (F12) и проверьте логи:
   - Должны быть сообщения "Инициализация формы..."
   - При отправке должны быть логи "Отправка данных формы..."
4. Проверьте терминал сервера - должны появляться логи при отправке форм
5. Используйте тестовую страницу `test-forms.html` для диагностики
